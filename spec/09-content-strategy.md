# Educational & Informational Content

Content strategy for helping members (and prospects) understand the club and use the facilities effectively. Most of this is CMS-managed content, not code.

---

## Range Guides

Each range (A through L) should have a dedicated page with:

### Standard Range Page Template

| Section | Content |
|---------|---------|
| Overview | What this range is for, photo(s) |
| Specifications | Distances, positions, target types |
| Allowed Firearms | What calibers/types permitted |
| Prohibited | What's NOT allowed |
| Target Rules | Height, placement, approved holders |
| Shooting Positions | Bench, prone, standing — what's allowed |
| Special Rules | Range-specific safety rules |
| Equipment Provided | Target frames, benches, etc. |
| What to Bring | Recommended gear |
| Tips | Best practices, etiquette |
| Photos | Multiple angles, example setups |

### Range-Specific Content

| Range | Name | Key Info |
|-------|------|----------|
| A | Pistol (25/50 yd) | Primarily pistol, .22 and muzzleloaders OK. No centerfire rifles except position 41. |
| B | Plinking (7-15 yd) | Rimfire and centerfire pistol only. No rifles. Paper targets only. |
| C | Benchrest/High Power (100/200 yd) | Centerfire rifles 2000+ FPS. No rimfire. Bench only at 100yd. |
| D | .22 Silhouette (40-100m) | .22 LR only. No magnums. Prone to standing OK. |
| E | Pistol Silhouette/Black Powder (40-100m) | All firearms. Shotgun patterning. Clay birds OK. |
| F | Air Gun (Indoor, 10m) | Air rifles/pistols only. Education Building. Requires ARO supervision. |
| G | CHL/Education | Reserved for CHL classes and approved education. |
| H-K | IDPA/IPSC Bays | Handguns only. Static firing positions unless during match. Paper targets only. |
| L | IDPA/IPSC Bay | Rifles and shotguns only (handguns during matches). |

- Descriptive title

- Open Graph image
- Structured data (LocalBusiness, Event, FAQ)

## Accessibility

All content should:

- Use proper heading hierarchy (h1 → h2 → h3)
- Include alt text for images
- Have sufficient color contrast
- Be readable at 200% zoom
- Work with screen readers
