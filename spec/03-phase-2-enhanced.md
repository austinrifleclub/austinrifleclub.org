# Phase 2: Enhanced Features (Should Have)

These features significantly improve the member experience and operational efficiency.

---

## 2.1 Digital Member Card

| Feature | Description |
|---------|-------------|
| QR code | Unique to member, scannable at gate |
| Apple Wallet / Google Wallet | Add card to phone |
| Family cards | Each family member has own card |
| Expiration | Shows on card, auto-updates |
| Offline access | Works without internet |

### Card Information

- Member name
- Badge number
- Member type (Individual, Family, Life, etc.)
- Photo (optional)
- Expiration date
- QR code for verification

### Wallet Integration

- Apple Wallet pass (.pkpass)
- Google Wallet pass
- Auto-update when membership renews
- Visual change when expired

## 2.2 Match Results & Statistics

### Match Scoring

| Feature | Description |
|---------|-------------|
| Score entry | Volunteers enter scores, or sync from Practiscore |
| Division | Track USPSA/IDPA/other divisions |
| Results page | Sortable by overall, division, class |
| Personal history | All your match results in one place |

### Statistics

| Stat | Description |
|------|-------------|
| Matches attended | Total count |
| Win rate | % of matches won in division |
| Average finish | Percentile |
| Personal bests | Best score, best stage |
| Improvement | Trend over time |

### Classifications

- Track USPSA classification (D, C, B, A, M, GM)
- Track IDPA classification
- Show on profile (optional)
- Historical progression

### Practiscore Integration

- Import match results automatically
- Map shooters to member accounts
- Calculate club-specific statistics
- Season standings

## 2.4 Classifieds

### Listing Types

| Type | Description |
|------|-------------|
| For Sale | Member selling item |
| Wanted | Member looking for item |
| Free | Giving away |

### Categories

- Handguns
- Rifles
- Shotguns
- Ammunition
- Reloading
- Optics
- Accessories
- Other

### Listing Fields

- Title
- Description
- Photos (up to 5)
- Price (or "Make Offer")
- Category
- Condition (New, Like New, Good, Fair)
- Contact preference (email, phone, message)
- Expiration (30 days default)

### Rules

- Members only
- Must follow all state/federal laws
- No prohibited items
- Club is not a party to any transaction
- Report button for violations

### Features

- Search and filter
- Save favorites
- Contact seller via site (hide email)
- Mark as sold
- Renew listing
- Admin moderation

## 2.5 Reservations & Rentals

### Range Reservations

| Field | Description |
|-------|-------------|
| Range | Which range (A-L) |
| Date | When |
| Time | Start and end |
| Purpose | Private practice, class, event |
| Attendees | Expected count |
| Cost | $X per hour or flat fee |

### Approval Workflow

1. Member submits request
2. Admin reviews
3. Approved → added to calendar, range marked reserved
4. Denied → member notified with reason

### Equipment Rentals

| Item | Cost | Deposit |
|------|------|---------|
| Shot timer | $5/day | None |
| Steel targets | $20/day | $50 |
| Target stands (set of 4) | $10/day | None |
| Classroom (education building) | $50/half day | None |

### Locker Rentals

- Annual fee: $50
- First come, first served
- Member provides own lock
- Contents are member's responsibility

### Rental Features

- Online booking calendar
- Availability display
- Payment at checkout
- Pickup/return instructions
- Late fee tracking
