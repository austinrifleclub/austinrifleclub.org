## 3.2 Mentorship Program

### How It Works

1. New member opts in to mentorship
2. Admin matches with experienced member
3. Both notified via email
4. Mentor reaches out
5. Mentor logs mentorship activities
6. After 90 days, both provide feedback

### Mentor Qualifications

- Member for 2+ years
- Attended 10+ matches
- Volunteer hours logged
- No safety violations

### Mentee Preferences

- Primary interest (pistol, rifle, shotgun, competition)
- Experience level
- Availability
- Goals

### Recognition

- Badge on profile
- Annual mentor appreciation
- Priority registration for events

## 3.3 Photo Gallery

### Albums

| Type | Created By |
|------|------------|

| Range photos | Admin uploads |
| Member uploads | Any member |

### Features

- Upload multiple photos
- Tag members in photos
- Download originals
- Slideshow view
- Photo of the month
- Thumbnail grid
- Lightbox view

### Rules

- No offensive content
- No photos of people without consent
- Admin can remove any photo
- Max file size: 10MB
- Supported formats: JPG, PNG, HEIC
