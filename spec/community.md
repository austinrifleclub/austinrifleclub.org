## Classifieds

### Listing Types

| Type | Description |
|------|-------------|
| For Sale | Member selling item |
| Wanted | Member looking for item |
| Free | Giving away |

### Categories

- Handguns
- Rifles
- Shotguns
- Ammunition
- Reloading
- Optics
- Accessories
- Other

### Listing Fields

| Field | Required | Notes |
|-------|----------|-------|
| title | Yes | Max 100 chars |
| description | Yes | Max 2000 chars |
| photos | No | Up to 5 images |
| price | No | Or "Make Offer" |
| category | Yes | From list |
| condition | Yes | New, Like New, Good, Fair |
| contact_preference | Yes | Email, phone, or message |
| expires_at | Auto | 30 days default |

### Rules

- Members only (active membership)
- Must follow all state/federal laws
- No prohibited items
- Club is not a party to any transaction
- Report button for violations

### Features

- Search and filter
- Save favorites
- Contact seller via site (hide email)
- Mark as sold
- Renew listing (resets 30 days)
- Admin moderation

## Mentorship Program

### How It Works

1. New member opts in to mentorship
2. Admin matches with experienced member
3. Both notified via email
4. Mentor reaches out
5. Mentor logs mentorship activities
6. After 90 days, both provide feedback

### Mentor Qualifications

| Requirement | Minimum |
|-------------|---------|
| Membership duration | 2+ years |
| Matches attended | 10+ |
| Volunteer hours | Some logged |
| Safety violations | None |

### Mentee Preferences

- Primary interest (pistol, rifle, shotgun, competition)
- Experience level
- Availability
- Goals

### Recognition

- Badge on mentor's profile
- Annual mentor appreciation
- Priority registration for events
